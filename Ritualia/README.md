# Ritualia

Ritualia es una aplicación web moderna orientada al comercio electrónico.

## Características Destacadas

*   **Subida de Archivos con Multer**: Integración de `multer` para gestionar y procesar eficientemente la carga de imágenes de los productos en la plataforma.
*   **Contexto de Cupones (Coupon Context)**: Manejo del estado global de los cupones de descuento utilizando React Context para facilitar su aplicación en el carrito de compras.
*   **Mensajes y Opiniones en Productos**: Sistema integrado que permite a los usuarios dejar sus opiniones y comentarios directamente en el detalle de cada artículo.
*   **Variedad de un Mismo Producto**: Visualización avanzada que muestra distintas variantes de un mismo producto (ej. distintos formatos, colores o aromas), facilitando la elección al usuario.
*   **HomePage Creada por un Agente de IA**: La página principal (*HomePage*), incluyendo su diseño general, ha sido desarrollada en colaboración con un agente de Inteligencia Artificial para lograr una estética moderna y funcional.
